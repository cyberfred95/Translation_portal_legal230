Comparer la base locale avec un fichier CSV

Format du fichier CSV

Le fichier CSV doit contenir les colonnes suivantes (dans l'ordre) :

  • file : Nom du fichier de glossaire (ex: ACCOUNTING_EN_FR.xlsx)
  • source_language : Langue source (code à 2 lettres, ex: EN)
  • target_language : Langue cible (code à 2 lettres, ex: FR)
  • domain : Domaine du glossaire (ex: Accounting)
  • user : (Optionnel) Username si le glossaire appartient à un utilisateur
  • group : (Optionnel) Nom du groupe si le glossaire appartient à un groupe

Note: C'est le même format que le fichier CSV utilisé pour le batch upload.
