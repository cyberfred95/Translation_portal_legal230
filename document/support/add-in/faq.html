Questions générales

Qu'est-ce que LEXA pour Microsoft Word ?

LEXA pour Microsoft Word est un complément (add-in) qui permet de traduire directement du texte dans vos documents Word. Il est spécialement conçu pour la traduction juridique et préserve le formatage de vos documents.

LEXA pour Word est-il gratuit ?

LEXA pour Word nécessite un compte LEXA actif et une clé API valide. L'utilisation du complément dépend de votre abonnement LEXA. Contactez-nous pour connaître les tarifs et options d'abonnement.

Quelles versions de Word sont compatibles ?

LEXA pour Word est compatible avec :

  • Microsoft Word 2016 ou version ultérieure (Windows)
  • Microsoft Word 2016 ou version ultérieure (Mac)
  • Word Online (version web)
  • Word pour iPad

Ai-je besoin d'une connexion Internet ?

Oui, LEXA pour Word nécessite une connexion Internet active pour communiquer avec les serveurs de traduction LEXA et récupérer les traductions.

Mes données sont-elles sécurisées ?

Oui, toutes les communications entre LEXA pour Word et les serveurs LEXA sont chiffrées via HTTPS. Votre clé API est stockée localement sur votre ordinateur. Consultez notre politique de confidentialité pour plus d'informations.

Installation et configuration

Comment installer LEXA pour Word ?

Il existe deux méthodes :

  1. Via le Microsoft Store : Ouvrez Word > Insertion > Compléments > Recherchez "LEXA" > Installer
  2. Installation manuelle : Utilisez le fichier manifest.xml fourni par votre administrateur (voir guide d'installation complet)

Où trouver ma clé API LEXA ?

Pour obtenir votre clé API :

  1. Connectez-vous à portail.lexamt.fr
  2. Accédez à "Paramètres" ou "Mon compte"
  3. Recherchez la section "Clé API"
  4. Copiez votre clé (une longue chaîne de caractères)

Comment configurer ma clé API dans le complément ?

Une fois LEXA installé :

  1. Cliquez sur le bouton LEXA dans Word
  2. Cliquez sur l'icône Paramètres (roue dentée)
  3. Collez votre clé API dans le champ prévu
  4. Cliquez sur Sauvegarder
  5. Testez la clé avec le bouton "Tester la clé API"

Je ne vois pas le bouton LEXA dans Word, que faire ?

Solutions possibles :

  • Redémarrez Word complètement
  • Vérifiez dans Fichier > Options > Compléments si LEXA n'est pas désactivé
  • Assurez-vous que l'installation s'est bien terminée
  • Vérifiez que vous utilisez Word 2016 ou ultérieur

Puis-je utiliser LEXA sur plusieurs ordinateurs ?

Oui, vous pouvez installer LEXA sur plusieurs ordinateurs. Vous devrez configurer votre clé API sur chaque installation. Votre clé API et votre compte LEXA peuvent être utilisés sur plusieurs appareils.

Utilisation

Comment traduire du texte avec LEXA ?

Processus en 5 étapes :

  1. Sélectionnez le texte dans votre document Word
  2. Choisissez les langues source et cible dans le panneau LEXA
  3. Sélectionnez un domaine de spécialisation (optionnel)
  4. Cliquez sur "Traduire"
  5. Cliquez sur "Remplacer" pour insérer la traduction

Le formatage de mon texte est-il préservé ?

Oui, LEXA préserve automatiquement :

  • Le gras, l'italique, le souligné et le barré
  • Les couleurs de texte
  • Les polices et tailles de caractères
  • Les listes à puces et numérotées
  • Les exposants et indices

Combien de texte puis-je traduire à la fois ?

Pour de meilleurs résultats, nous recommandons de traduire :

  • Paragraphe par paragraphe pour des documents longs
  • Section par section pour des documents moyens
  • La limite exacte dépend de votre abonnement LEXA

Traduire de plus petits segments améliore la qualité et la rapidité.

Quelle est la différence entre "Remplacer" et "Insérer" ?

Actuellement, LEXA propose principalement le mode "Remplacer" :

  • Remplacer : Le texte original sélectionné est remplacé par la traduction
  • Insérer : La traduction est ajoutée après le texte original (fonctionnalité en développement)

Conseil : Avant de remplacer, faites une copie de votre texte original dans un autre document si vous souhaitez le conserver.

Puis-je annuler une traduction ?

Oui, utilisez simplement la fonction "Annuler" de Word (Ctrl+Z ou Cmd+Z) immédiatement après avoir inséré une traduction. Cela restaurera votre texte original.

Les traductions sont-elles sauvegardées quelque part ?

Les traductions ne sont pas automatiquement sauvegardées par le complément. Une fois insérée dans votre document, la traduction fait partie du document Word que vous devez sauvegarder normalement.

Comment changer la langue de l'interface LEXA ?

Pour changer la langue de l'interface :

  1. Cliquez sur le sélecteur de langue (drapeau) en haut à droite du panneau LEXA
  2. Sélectionnez votre langue préférée parmi : Français, Anglais, Espagnol, Allemand, Italien
  3. L'interface se met à jour immédiatement

Qu'est-ce qu'un domaine de spécialisation et dois-je en choisir un ?

Les domaines de spécialisation sont des catégories juridiques qui améliorent la précision de la traduction :

  • Exemples : Droit des contrats, Propriété intellectuelle, Droit du travail, etc.
  • Choisir un domaine adapte la terminologie à votre type de document
  • Ce n'est pas obligatoire mais fortement recommandé pour des traductions juridiques

Langues et traduction

Quelles langues puis-je traduire ?

Les langues disponibles dépendent de votre abonnement LEXA. Les langues sont automatiquement chargées depuis votre compte lorsque vous ouvrez le complément. Contactez le support LEXA pour connaître les paires de langues disponibles dans votre abonnement.

Puis-je traduire dans les deux sens (ex: FR→EN et EN→FR) ?

Oui, absolument. Utilisez le bouton d'inversion (⇄) entre les sélecteurs de langues pour échanger rapidement la langue source et la langue cible.

LEXA détecte-t-il automatiquement la langue du texte ?

Non, vous devez sélectionner manuellement la langue source. Cependant, LEXA se souvient de vos derniers choix de langues et les préselectionne automatiquement lors de votre prochaine utilisation.

Les langues que je veux ne s'affichent pas, pourquoi ?

Causes possibles :

  • Votre clé API n'est pas configurée ou est invalide
  • Problème de connexion Internet
  • Les langues ne sont pas incluses dans votre abonnement
  • Erreur temporaire du serveur

Solution : Testez votre clé API dans les Paramètres et vérifiez votre connexion. Contactez le support si le problème persiste.

Problèmes techniques

Le bouton "Traduire" est grisé, je ne peux pas cliquer dessus

Le bouton "Traduire" est désactivé si :

  • Aucun texte n'est sélectionné dans le document
  • La sélection est vide
  • Aucune clé API n'est configurée

Solution : Sélectionnez du texte dans votre document et vérifiez que votre clé API est bien configurée.

J'obtiens une erreur "Clé API invalide"

Solutions :

  1. Vérifiez que vous avez copié la clé complète sans espaces superflus
  2. Assurez-vous que votre abonnement LEXA est actif
  3. Générez une nouvelle clé API depuis votre compte LEXA
  4. Contactez le support si le problème persiste

Le panneau LEXA ne s'ouvre pas ou affiche une page blanche

Solutions :

  • Vérifiez votre connexion Internet
  • Fermez et rouvrez Word complètement
  • Vérifiez que votre pare-feu autorise les connexions à portail.lexamt.fr
  • Videz le cache de Word et réessayez
  • Réinstallez le complément si nécessaire

La traduction est très lente

Optimisations possibles :

  • Traduisez des segments plus courts
  • Vérifiez la vitesse de votre connexion Internet
  • Fermez d'autres applications pour libérer des ressources
  • Évitez de traduire pendant les heures de forte affluence

J'obtiens une erreur 429 "Trop de requêtes"

Cette erreur signifie que vous avez atteint la limite de traductions autorisées dans un laps de temps donné. Attendez quelques minutes avant de réessayer. Si cela se produit fréquemment, contactez le support pour discuter de votre volume de traduction et des options d'abonnement.

La traduction insérée ne garde pas le formatage

Vérifications :

  • Assurez-vous d'utiliser une version récente de Word
  • Vérifiez que le formatage utilisé fait partie des formats supportés
  • Un formatage très complexe peut parfois ne pas être entièrement préservé
  • Signalez le problème au support avec un exemple si cela persiste

Puis-je utiliser LEXA hors ligne ?

Non, LEXA pour Word nécessite une connexion Internet pour fonctionner car les traductions sont effectuées sur les serveurs LEXA. Aucune traduction hors ligne n'est possible.

Sécurité et confidentialité

Où est stockée ma clé API ?

Votre clé API est stockée localement dans le stockage local (localStorage) de votre navigateur, sur votre ordinateur. Elle n'est jamais envoyée à des tiers et est uniquement utilisée pour authentifier vos requêtes vers les serveurs LEXA.

Mon texte traduit est-il stocké sur les serveurs LEXA ?

Les textes sont traités sur les serveurs LEXA pour effectuer la traduction, mais la politique de conservation dépend de votre contrat. Consultez la politique de confidentialité LEXA ou contactez le support pour plus de détails.

Puis-je révoquer ma clé API ?

Oui, connectez-vous à votre compte LEXA sur portail.lexamt.fr et générez une nouvelle clé API. L'ancienne sera automatiquement révoquée. Pensez ensuite à mettre à jour la clé dans le complément Word.

LEXA est-il conforme au RGPD ?

LEXA s'engage à respecter le Règlement Général sur la Protection des Données (RGPD). Pour plus d'informations sur la conformité et la protection des données, consultez notre politique de confidentialité sur portail.lexamt.fr.

Comment fonctionne la facturation de LEXA ?

La facturation dépend de votre plan d'abonnement LEXA. Connectez-vous à votre compte sur portail.lexamt.fr pour voir les détails de votre abonnement, votre usage, et vos options de facturation.

Y a-t-il une limite au nombre de traductions ?

Cela dépend de votre plan d'abonnement LEXA. Certains plans ont des limites mensuelles, d'autres sont illimités. Vérifiez les détails de votre plan dans votre compte LEXA.

Puis-je essayer LEXA gratuitement ?

Contactez le support LEXA pour connaître les options d'essai gratuit ou de démonstration disponibles.

Comment mettre à niveau mon abonnement ?

Pour changer de plan ou mettre à niveau votre abonnement, connectez-vous à portail.lexamt.fr et accédez à la section "Abonnement" ou contactez le support commercial.
