Présentation de LEXA pour Microsoft Word

LEXA pour Microsoft Word est un complément (add-in) qui vous permet de traduire directement du texte dans vos documents Word, sans avoir à quitter l'application. Conçu spécialement pour la traduction juridique, LEXA préserve la mise en forme de votre texte et vous offre des traductions de qualité professionnelle.

Avantages de LEXA pour Word

  • Traduction directe dans Word : Traduisez votre texte sans quitter votre document
  • Préservation de la mise en forme : Le formatage (gras, italique, couleurs, polices, listes) est conservé après traduction
  • Traduction spécialisée : Choix de domaines juridiques spécialisés pour des traductions précises
  • Support multilingue : Interface disponible en 5 langues (français, anglais, espagnol, allemand, italien)
  • Prévisualisation : Visualisez la traduction avant de l'insérer dans votre document
  • Flexibilité d'insertion : Choisissez de remplacer le texte original ou de l'insérer à côté

Versions de Word compatibles

  • Word 2016 ou version ultérieure (Windows et Mac)
  • Word Online
  • Word pour iPad

Prérequis

Avant d'installer LEXA pour Microsoft Word, assurez-vous de disposer des éléments suivants :

Logiciels requis

  • Microsoft Word 2016 ou version ultérieure ou Word Online
  • Une connexion Internet active
  • Un navigateur web moderne (Chrome, Edge, Firefox, Safari)

Compte LEXA

  • La clé API vou sest fournie sur demande à l'adresse support@lexa.com.t.fr
  • Une clé API LEXA valide (fournie dans votre compte)

Comment obtenir votre clé API ?

  1. Connectez-vous à votre compte LEXA sur portail.lexamt.fr
  2. Accédez à la section "Paramètres" ou "Mon compte"
  3. Recherchez la section "Clé API" ou "API Key"
  4. Copiez votre clé API (une chaîne de caractères longue et aléatoire)
  5. Conservez cette clé en lieu sûr, vous en aurez besoin pour configurer le complément

Installation de LEXA pour Word

Option 1 : Installation depuis le Microsoft Store

  1. Ouvrez Microsoft Word
  2. Cliquez sur l'onglet Insertion dans le ruban
  3. Cliquez sur Compléments ou Applications
  4. Dans la fenêtre qui s'ouvre, recherchez "LEXA"
  5. Cliquez sur Ajouter ou Installer
  6. Suivez les instructions à l'écran pour terminer l'installation

Option 2 : Installation manuelle du manifeste

Si votre administrateur système vous a fourni un fichier de manifeste (fichier .xml), suivez ces étapes :

Sur Windows

  1. Ouvrez Microsoft Word
  2. Accédez à Fichier > Options > Centre de gestion de la confidentialité
  3. Cliquez sur Paramètres du Centre de gestion de la confidentialité
  4. Sélectionnez Catalogues de compléments approuvés
  5. Dans le champ URL du catalogue, entrez le chemin vers le dossier contenant le manifeste
  6. Cochez Afficher dans le menu
  7. Cliquez sur OK
  8. Redémarrez Word
  9. Le complément LEXA devrait maintenant apparaître dans l'onglet Accueil

Sur Mac

  1. Ouvrez le Finder
  2. Appuyez sur Cmd+Maj+G pour ouvrir "Aller au dossier"
  3. Entrez le chemin : /Users/[VotreNom]/Library/Containers/com.microsoft.Word/Data/Documents/wef
  4. Créez le dossier wef s'il n'existe pas
  5. Copiez le fichier manifest.xml dans ce dossier
  6. Redémarrez Word
  7. Le complément LEXA devrait maintenant apparaître dans l'onglet Accueil

Sur Word Online

  1. Ouvrez un document dans Word Online
  2. Cliquez sur Insertion > Compléments
  3. Sélectionnez TÉLÉCHARGER MON COMPLÉMENT
  4. Cliquez sur Parcourir et sélectionnez le fichier manifest.xml
  5. Cliquez sur Télécharger

Vérification de l'installation

Pour vérifier que LEXA est bien installé :

  1. Ouvrez Microsoft Word
  2. Recherchez le bouton LEXA dans l'onglet Accueil du ruban
  3. Cliquez sur ce bouton : le panneau LEXA devrait s'ouvrir sur le côté droit de votre document

Configuration de LEXA

Avant de pouvoir utiliser LEXA pour la première fois, vous devez configurer votre clé API.

Configuration initiale de la clé API

  1. Ouvrez Microsoft Word et cliquez sur le bouton LEXA dans le ruban
  2. Le panneau LEXA s'ouvre sur le côté droit
  3. Vous verrez un message indiquant "Aucune clé API n'est enregistrée"
  4. Cliquez sur l'icône Paramètres (roue dentée) en haut à droite du panneau
  5. Dans le champ Clé API, collez votre clé API LEXA
  6. Cliquez sur le bouton Sauvegarder (icône de disquette)
  7. Un message de confirmation devrait apparaître

Test de la clé API

Pour vérifier que votre clé API fonctionne correctement :

  1. Dans la vue Paramètres, après avoir sauvegardé votre clé API
  2. Cliquez sur le bouton Tester la clé API
  3. Si la clé est valide, vous verrez un message de succès en vert
  4. Si la clé est invalide, vous verrez un message d'erreur en rouge
  5. En cas d'erreur, vérifiez que vous avez bien copié la clé complète sans espaces supplémentaires

Modification de la clé API

Si vous devez changer votre clé API (par exemple, pour des raisons de sécurité) :

  1. Cliquez sur l'icône Paramètres dans le panneau LEXA
  2. Le champ affiche votre clé actuelle (masquée par des points)
  3. Supprimez l'ancienne clé et collez la nouvelle
  4. Cliquez sur Sauvegarder
  5. Testez la nouvelle clé avec le bouton Tester la clé API

Choix de la langue de l'interface

LEXA supporte 5 langues d'interface : français, anglais, espagnol, allemand et italien.

  1. Par défaut, LEXA détecte la langue de votre navigateur
  2. Pour changer la langue, cliquez sur le sélecteur de langue (drapeau) en haut à droite du panneau
  3. Sélectionnez la langue souhaitée dans le menu déroulant
  4. L'interface se met à jour immédiatement dans la langue choisie
  5. Votre choix est sauvegardé automatiquement

Utilisation de LEXA

Une fois configuré, LEXA est prêt à traduire vos documents. Voici comment l'utiliser.

Traduction de texte

Étape 1 : Sélectionner le texte à traduire

  1. Dans votre document Word, sélectionnez le texte que vous souhaitez traduire
  2. Le texte peut contenir du formatage (gras, italique, couleurs, listes, etc.)
  3. Vous pouvez sélectionner un mot, une phrase, un paragraphe ou plusieurs paragraphes

Étape 2 : Choisir les langues

  1. Dans le panneau LEXA, sélectionnez la langue source (langue du texte à traduire)
  2. Sélectionnez la langue cible (langue vers laquelle traduire)
  3. Vous pouvez inverser les langues source et cible en cliquant sur le bouton ⇄
  4. Vos choix de langues sont sauvegardés automatiquement pour la prochaine fois

Étape 3 : Choisir un domaine (optionnel)

  1. Dans le menu déroulant Domaine, sélectionnez un domaine juridique spécialisé
  2. Les domaines disponibles sont chargés depuis votre compte LEXA
  3. Choisir un domaine améliore la précision de la traduction pour ce type de texte
  4. Votre choix de domaine est également sauvegardé

Étape 4 : Lancer la traduction

  1. Cliquez sur le bouton Traduire
  2. Une animation de chargement apparaît pendant la traduction
  3. La traduction s'affiche dans la zone de prévisualisation en dessous
  4. Le formatage du texte original est préservé dans la traduction

Étape 5 : Insérer la traduction

Une fois la traduction affichée, vous avez deux options :

  • Remplacer : Cliquez sur le bouton Remplacer pour remplacer le texte sélectionné par la traduction
  • Insérer : La traduction est insérée juste après votre sélection, sans supprimer le texte original

Conseils d'utilisation

  • Traduisez par segments : Pour de meilleurs résultats, traduisez paragraphe par paragraphe plutôt que des documents entiers
  • Vérifiez la prévisualisation : Avant d'insérer, vérifiez toujours la traduction dans la zone de prévisualisation
  • Conservez le formatage : LEXA préserve automatiquement le gras, l'italique, les couleurs et les listes
  • Utilisez les domaines : Pour des traductions juridiques, sélectionnez toujours le domaine approprié
  • Sauvegardez régulièrement : Pensez à enregistrer votre document Word régulièrement pendant la traduction

Fonctionnalités détaillées

Préservation du formatage

LEXA préserve automatiquement les éléments de mise en forme suivants lors de la traduction :

  • Styles de texte : Gras, italique, souligné, barré
  • Police : Nom de la police et taille
  • Couleurs : Couleur du texte
  • Listes : Listes à puces et listes numérotées, avec niveaux d'indentation
  • Exposants et indices : Pour les notes de bas de page, formules chimiques, etc.

Langues supportées

LEXA supporte de nombreuses paires de langues. Les langues disponibles dépendent de votre abonnement et sont chargées automatiquement depuis votre compte LEXA.

Interface multilingue

L'interface de LEXA est disponible en 5 langues :

  • Français (fr)
  • Anglais (en)
  • Espagnol (es)
  • Allemand (de)
  • Italien (it)

Domaines de spécialisation

LEXA offre des traductions spécialisées dans différents domaines juridiques. Les domaines disponibles dépendent de votre abonnement et incluent typiquement :

  • Droit des contrats
  • Droit de la propriété intellectuelle
  • Droit des sociétés
  • Droit du travail
  • Droit pénal
  • Et bien d'autres...

Stockage des préférences

LEXA sauvegarde automatiquement vos préférences localement sur votre ordinateur :

  • Votre clé API (stockée de manière sécurisée)
  • Vos dernières langues source et cible sélectionnées
  • Votre dernier domaine sélectionné
  • Votre langue d'interface préférée

Ces préférences sont conservées entre les sessions, vous n'avez donc pas besoin de les reconfigurer à chaque fois.

Prévisualisation de la traduction

La zone de prévisualisation vous permet de :

  • Visualiser la traduction avant de l'insérer dans le document
  • Vérifier que le formatage est correctement préservé
  • Relancer une nouvelle traduction si le résultat ne vous convient pas
  • Voir la traduction avec son formatage HTML préservé

Dépannage

Cette section vous aide à résoudre les problèmes courants que vous pourriez rencontrer avec LEXA pour Word.

Le complément LEXA n'apparaît pas dans Word

Vérifications à effectuer :

  1. Vérifiez l'installation : Assurez-vous que le complément est bien installé (voir section Installation)
  2. Redémarrez Word : Fermez complètement Word et rouvrez-le
  3. Vérifiez les compléments désactivés :
    • Allez dans Fichier > Options > Compléments
    • En bas de la fenêtre, sélectionnez Compléments désactivés dans la liste déroulante
    • Cliquez sur Atteindre
    • Si LEXA apparaît, sélectionnez-le et cliquez sur Activer
  4. Vérifiez la version de Word : LEXA nécessite Word 2016 ou ultérieur

Message "Aucune clé API n'est enregistrée"

Solution :

  1. Cliquez sur l'icône Paramètres (roue dentée)
  2. Entrez votre clé API dans le champ prévu
  3. Cliquez sur Sauvegarder
  4. Si vous n'avez pas de clé API, connectez-vous à portail.lexamt.fr pour l'obtenir

Erreur "Clé API invalide" lors du test

Solutions possibles :

  1. Vérifiez la clé : Assurez-vous d'avoir copié la clé complète sans espaces au début ou à la fin
  2. Vérifiez l'expiration : Votre clé API peut avoir expiré, connectez-vous à votre compte LEXA pour vérifier
  3. Vérifiez votre abonnement : Assurez-vous que votre abonnement LEXA est actif
  4. Générez une nouvelle clé : Si nécessaire, générez une nouvelle clé API depuis votre compte LEXA

Le bouton "Traduire" est désactivé (grisé)

Causes possibles :

  • Aucun texte sélectionné : Vous devez d'abord sélectionner du texte dans votre document Word
  • Sélection vide : Assurez-vous que votre sélection contient du texte
  • Aucune clé API : Vérifiez que votre clé API est bien configurée

Le panneau LEXA ne se charge pas ou affiche une erreur

Solutions :

  1. Vérifiez votre connexion Internet : LEXA nécessite une connexion active
  2. Vérifiez les pare-feu/proxy : Assurez-vous que votre pare-feu ou proxy d'entreprise autorise les connexions à portail.lexamt.fr
  3. Videz le cache :
    • Fermez Word
    • Supprimez les fichiers temporaires de Word
    • Rouvrez Word
  4. Réinstallez le complément : Si le problème persiste, désinstallez et réinstallez LEXA

Les langues ou domaines ne se chargent pas

Vérifications :

  1. Vérifiez la clé API : Testez votre clé API dans les Paramètres
  2. Vérifiez votre connexion : Assurez-vous d'être connecté à Internet
  3. Cliquez sur Actualiser : Si un bouton "Actualiser" apparaît, cliquez dessus pour recharger les données
  4. Vérifiez votre abonnement : Certains domaines peuvent nécessiter un abonnement spécifique

La traduction ne préserve pas le formatage

Points à vérifier :

  • Formatage supporté : Vérifiez que le formatage que vous utilisez fait partie des formats supportés (voir section Fonctionnalités)
  • Complexité du formatage : Un formatage très complexe peut parfois ne pas être complètement préservé
  • Version de Word : Assurez-vous d'utiliser une version récente de Word

Erreur "Nous ne pouvons pas ouvrir ce complément"

Sur Word de bureau :

  1. Cette erreur indique généralement un problème de certificat HTTPS
  2. Si vous utilisez une version de développement, assurez-vous que le serveur local est en cours d'exécution
  3. Vérifiez que le certificat SSL est approuvé
  4. Contactez votre administrateur système si vous êtes en environnement professionnel

Le texte traduit ne s'insère pas dans le document

Solutions :

  1. Vérifiez la sélection : Assurez-vous qu'une zone de texte est toujours sélectionnée dans Word
  2. Vérifiez les permissions : Le document n'est pas en lecture seule
  3. Essayez de cliquer à nouveau : Parfois une seconde tentative fonctionne
  4. Relancez la traduction : Cliquez à nouveau sur "Traduire" puis sur "Remplacer"

Message d'erreur lors de la traduction

Types d'erreurs et solutions :

  • Erreur 401 (Non autorisé) : Votre clé API est invalide ou expirée, vérifiez-la dans les Paramètres
  • Erreur 403 (Accès interdit) : Vous n'avez pas accès à cette fonctionnalité, vérifiez votre abonnement
  • Erreur 429 (Trop de requêtes) : Vous avez atteint la limite de traductions, attendez quelques minutes
  • Erreur 500 (Erreur serveur) : Problème temporaire du serveur, réessayez dans quelques minutes
  • Erreur réseau : Vérifiez votre connexion Internet

Problèmes de performances (traduction lente)

Optimisations possibles :

  • Traduisez des segments plus courts : Divisez les longs textes en paragraphes plus petits
  • Vérifiez votre connexion : Une connexion lente affecte la vitesse de traduction
  • Fermez d'autres applications : Libérez des ressources système
  • Simplifiez le formatage : Un formatage très complexe peut ralentir le processus
