Commencer avec LEXA en 5 minutes

Ce guide vous aidera à installer et utiliser LEXA pour Word rapidement. Pour des informations détaillées, consultez le guide complet.

Étape 1 : Installer LEXA pour Word

Installation depuis le Microsoft Store (Recommandé)

  1. Ouvrez Microsoft Word
  2. Cliquez sur Insertion dans le menu
  3. Cliquez sur Compléments ou Applications
  4. Recherchez "LEXA"
  5. Cliquez sur Ajouter

Alternative : Installation manuelle

Si votre administrateur vous a fourni un fichier manifest.xml, suivez les instructions d'installation manuelle dans le guide complet.

Vérification

Après l'installation, vous devriez voir un bouton LEXA dans l'onglet Accueil de Word.

Étape 2 : Obtenir votre clé API

  1. Ouvrez votre navigateur et allez sur portail.lexamt.fr
  2. Connectez-vous avec votre compte LEXA
  3. Accédez à Paramètres ou Mon compte
  4. Recherchez la section "Clé API"
  5. Copiez votre clé API (une longue chaîne de caractères)

Note : Si vous n'avez pas de compte LEXA, contactez support@lexamt.fr pour en créer un.

Étape 3 : Configurer LEXA dans Word

Ouvrir le panneau LEXA

  1. Dans Word, cliquez sur le bouton LEXA dans le ruban
  2. Le panneau LEXA s'ouvre sur le côté droit

Entrer votre clé API

  1. Dans le panneau LEXA, cliquez sur l'icône Paramètres (roue dentée en haut à droite)
  2. Collez votre clé API dans le champ "Clé API"
  3. Cliquez sur le bouton Sauvegarder (icône de disquette)
  4. Cliquez sur Tester la clé API pour vérifier qu'elle fonctionne

Résultat attendu

Si la clé est valide, vous verrez un message de succès en vert. Sinon, vérifiez que vous avez bien copié la clé complète.

Étape 4 : Première traduction

Préparation

  1. Retournez à la vue principale en cliquant sur la flèche de retour (ou fermez les Paramètres)
  2. Dans votre document Word, sélectionnez du texte à traduire

Configuration de la traduction

  1. Dans le panneau LEXA, sélectionnez la langue source (ex: Français)
  2. Sélectionnez la langue cible (ex: Anglais)
  3. Sélectionnez un domaine si votre texte est spécialisé (optionnel mais recommandé)

Traduction

  1. Cliquez sur le bouton Traduire
  2. Attendez quelques secondes pendant le traitement
  3. La traduction apparaît dans la zone de prévisualisation

Insertion

  1. Vérifiez la traduction dans la zone de prévisualisation
  2. Cliquez sur Remplacer pour remplacer le texte original par la traduction

Conseil : Utilisez Ctrl+Z (Windows) ou Cmd+Z (Mac) pour annuler si nécessaire.

Astuces pour bien démarrer

Préservation du formatage

LEXA préserve automatiquement le formatage de votre texte :

  • Gras, italique, souligné
  • Couleurs de texte
  • Polices et tailles
  • Listes à puces et numérotées

Inverser les langues

Utilisez le bouton ⇄ entre les sélecteurs de langues pour échanger rapidement la langue source et la langue cible.

Changer la langue de l'interface

Cliquez sur le sélecteur de langue (drapeau) en haut à droite pour choisir parmi 5 langues : Français, Anglais, Espagnol, Allemand, Italien.

Traduire efficacement

  • Traduisez paragraphe par paragraphe pour de meilleurs résultats
  • Sélectionnez toujours un domaine adapté pour des traductions juridiques précises
  • Vérifiez la prévisualisation avant d'insérer

Mémorisation des préférences

LEXA sauvegarde automatiquement :

  • Votre clé API
  • Vos dernières langues sélectionnées
  • Votre dernier domaine choisi
  • Votre langue d'interface

Vous n'avez donc pas à reconfigurer à chaque utilisation !

Vos premiers pas avec LEXA

Exercice 1 : Traduire un paragraphe simple

  1. Tapez ou collez un paragraphe en français dans Word
  2. Sélectionnez-le
  3. Dans LEXA : Français → Anglais
  4. Cliquez sur "Traduire"
  5. Vérifiez la prévisualisation
  6. Cliquez sur "Remplacer"

Exercice 2 : Traduire avec formatage

  1. Tapez un paragraphe avec du texte en gras et de l'italique
  2. Sélectionnez et traduisez
  3. Vérifiez que le formatage est préservé dans la traduction

Exercice 3 : Utiliser un domaine spécialisé

  1. Sélectionnez un texte juridique
  2. Choisissez le domaine approprié (ex: "Droit des contrats")
  3. Traduisez et comparez avec une traduction sans domaine

Exercice 4 : Inverser les langues

  1. Configurez Français → Anglais
  2. Cliquez sur le bouton ⇄
  3. Vérifiez que les langues sont maintenant Anglais → Français
  4. Traduisez un texte anglais en français

Problèmes courants au démarrage

Le bouton LEXA n'apparaît pas

Solution : Redémarrez Word complètement. Si le problème persiste, vérifiez l'installation.

Le test de clé API échoue

Solution : Vérifiez que vous avez copié la clé complète sans espaces. Assurez-vous que votre abonnement LEXA est actif.

Les langues ne se chargent pas

Solution : Vérifiez votre connexion Internet. Testez votre clé API. Cliquez sur "Actualiser" si un bouton est disponible.

Le bouton "Traduire" est grisé

Solution : Assurez-vous d'avoir sélectionné du texte dans votre document Word.

La traduction est lente

Solution : Traduisez des segments plus courts. Vérifiez votre connexion Internet.

Pour plus d'aide : Consultez la FAQ ou le guide de dépannage complet.
